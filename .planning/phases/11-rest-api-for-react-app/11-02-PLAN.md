  <action>
    Create SessionViewSet per RESEARCH.md "Pattern 2: ReadOnlyModelViewSet for Dashboard" and "DRF ViewSet with Optimization":
    
    ```python
    from rest_framework import viewsets, permissions
    from api.v1.serializers.session import SessionSerializer
    from deep_agent.models import AgentIssueSession
    
    class SessionViewSet(viewsets.ReadOnlyModelViewSet):
        """
        Expose agent session states to React dashboard.
        Requires JWT authentication for all operations.
        Optimized to prevent N+1 queries.
        """
        queryset = AgentIssueSession.objects.all().order_by('-created_at')
        serializer_class = SessionSerializer
        permission_classes = [permissions.IsAuthenticated]
        
        def get_queryset(self):
            # Prevent N+1: prefetch related Plane issue data
            return super().get_queryset().select_related(
                'plane_issue'
            ).prefetch_related(
                'langfuse_traces'
            )
    ```
    
    Verify field names exist on AgentIssueSession model before using in select_related/prefetch_related.

| Boundary | Description |
|----------|-------------|
| API endpoint → Database | Untrusted queries — pagination, field restrictions prevent data exposure |
| Serializer → Response | Output validation — read_only_fields prevent accidental mutations |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-11-06 | Information Disclosure | Session data exposure | mitigate | ReadOnlyModelViewSet with explicit fields list, read_only_fields prevents mutations |
